#  視訊聊天畫面 Layout 練習
✨ 這個分支是練習用的，跟主要的版本差別只有把「VideoChatLayout」相關的做法都刪掉。

💻 你需要自己完成的地方都有加上 **TODO**，總共**只有 ３ 個地方需要更改**。

## 作業簡介
設計視訊聊天畫面（其實都是靜態圖片），圖片大小會依據是否有被 pin 或是被隱藏而調整。
